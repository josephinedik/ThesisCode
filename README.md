## How to generate the results of Sections 8.4 & 8.5
The results of section 8.4 and 8.5 are generated by the code in the folder DutyOfCare. 

The results of section 8.4.1 can be generated using the following command:

`clingo Table1.lp Table2.lp Table3.lp Table4.lp Table5.lp` 

The results of section 8.4.2 can be generated using the following command:

`clingo Table1.lp Table2.lp Table3.lp Table4.lp Table5.lp Case1.lp`

The results of section 8.4.3 can be generated using the following command:

`clingo Table1.lp Table2.lp Table3.lp Table4.lp Table5.lp Case2.lp`

The scalability results of Section 8.5 can be generated using the following command: 

`clingo Table1.lp Table2.lp Table3.lp Table4.lp Table5.lp Scalability.lp`


## How to generate the results of Section 8.7
The results of this section are generated by the code in the folder Explanations. 

For the full analysis of the first judge, run the following command:

`python analyzejudgeone.py`

For the full analysis of the appeal judge, run the following command:

`python analyzeappealjudge.py`
